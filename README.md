# MockMate — Frontend

React + Redux frontend for MockMate, an AI-powered mock interview platform.

## Live Demo
https://mock-mate-frontend-gilt.vercel.app

## Tech Stack
- React, Redux Toolkit, React Router
- Axios, STOMP/SockJS for WebSocket
- Web Speech API for voice input
- Recharts for analytics visualization
- Deployed on Vercel

## Running locally
\`\`\`bash
git clone https://github.com/yourusername/MockMate-Front.git
cd frontend
npm install
npm run dev
\`\`\`
